# music-trainer-web

`music-trainer-web` 是一个用于单音听辨和五线谱识读练习的浏览器工具。

## 功能

| 模块 | 说明 |
|---|---|
| 单音听辨 | 播放参考音域和随机单音，用户判断听到的音名 |
| 五线谱练习 | 显示两个五线谱音符，用户判断两个音名和音程关系 |

## 当前能力

| 模块 | 已支持内容 |
|---|---|
| 单音听辨 | 支持三角钢琴和原声吉他两种在线采样音色 |
| 单音听辨 | 支持 C3-B3、C4-B4、C5-B5 三个音域 |
| 单音听辨 | 支持开始、停止、重播参考音域、重播单音和会话统计 |
| 五线谱练习 | 支持高音、低音、中音、次中音四种谱号 |
| 五线谱练习 | 支持入门、基础、进阶、挑战四个难度等级 |
| 五线谱练习 | 支持两音播放、自动播放开关和三组四选一答题 |

## 技术栈

| 类型 | 使用内容 |
|---|---|
| 前端 | React、TypeScript |
| 构建 | Vite |
| 测试 | Vitest、Testing Library |
| 五线谱渲染 | VexFlow |
| 音频 | Web Audio API、在线采样资源 |

## 本地运行

需要先安装 Node.js，建议使用 Node.js 20 或更新版本。

```bash
npm install
npm run dev
```

运行测试：

```bash
npm test -- --run
```

生成生产构建：

```bash
npm run build
```
