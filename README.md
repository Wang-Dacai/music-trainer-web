# music-trainer-web

`music-trainer-web` 是一个用于单音听辨、五线谱识读、节奏参考和吉他和弦识读的浏览器工具。

## 功能

| 模块 | 说明 |
|---|---|
| 单音听辨 | 播放参考大调音阶和随机目标音，用户判断目标音在音阶中的位置 |
| 五线谱练习 | 显示两个五线谱音符，用户判断两个音名和音程关系 |
| 节奏参考 | 浏览常见节奏型，查看吉他扫弦六线谱和五线谱，并用钢琴、吉他等音色播放示范 |
| 吉他和弦识读 | 搜索常见吉他和弦，查看组成音、声音试听、五线谱写法和常用指法 |

## 当前能力

| 模块 | 已支持内容 |
|---|---|
| 单音听辨 | 支持三角钢琴、原声吉他、尼龙吉他、小提琴、长笛、单簧管、风琴等在线采样音色 |
| 单音听辨 | 支持 C、G、F、D、Bb、A、Eb、E、Ab 常用大调音阶 |
| 单音听辨 | 支持低音区、中心区、高音区三个音域，默认使用中心区 |
| 单音听辨 | 支持音名、唱名、级数三种答案按钮显示方式，并允许同时显示其中两种 |
| 单音听辨 | 按“七个上行音阶音、主音、目标音”的顺序播放 |
| 单音听辨 | 支持开始、停止、重播参考音阶、重播单音和会话统计，停止后写入练习记录并清空本轮统计 |
| 五线谱练习 | 支持高音、低音、中音、次中音四种谱号 |
| 五线谱练习 | 支持入门、基础、进阶、挑战四个难度等级 |
| 五线谱练习 | 支持两音播放、自动播放开关和三组四选一答题，结束练习后写入练习记录并清空本轮统计 |
| 节奏参考 | 支持 4/4、3/4、2/4 拍号和 60、72、84、96、108、120 BPM 速度选项 |
| 节奏参考 | 支持基础拍点、八分细分、十六分细分、切分节奏和鼓手常用五类谱例 |
| 节奏参考 | 支持常见复杂节奏型，包括 Charleston、3-3-2 / Tresillo、Backbeat、Funk 十六分骨架和 Bossa 单小节骨架 |
| 节奏参考 | 支持把节奏型套入 C 大调上行、C 大调分解和弦、A 小调五声和流行短句等旋律 |
| 节奏参考 | 支持原声吉他、三角钢琴、尼龙吉他和管风琴等在线采样音色示范 |
| 节奏参考 | 支持预备拍、播放遍数选择、吉他扫弦六线谱和五线谱双谱面展示；当前不包含网页打拍交互和会话记录 |
| 吉他和弦识读 | 支持自然音和常见升降根音的大三、小三、属七、大七、小七、挂二、挂四和弦 |
| 吉他和弦识读 | 支持按和弦符号、等音别名和中文名称搜索，单根音搜索优先匹配和弦名称 |
| 吉他和弦识读 | 支持整和弦与分解和弦两种声音试听方式 |
| 吉他和弦识读 | 支持组成音说明、高音谱号和弦记谱，以及每个和弦至少两种常用吉他指法 |

## 技术栈

| 类型 | 使用内容 |
|---|---|
| 前端 | React、TypeScript |
| 构建 | Vite |
| 测试 | Vitest、Testing Library |
| 五线谱渲染 | VexFlow |
| 节奏图 / 和弦图 | VexFlow、SVG、CSS Grid |
| 音频 | Web Audio API、在线采样资源、旋律化节奏播放 |

## 本地运行

需要先安装 Node.js，建议使用 Node.js 20 或更新版本。

```bash
npm install
npm run dev
```

运行测试：

```bash
npm test -- --run
```

生成生产构建：

```bash
npm run build
```
